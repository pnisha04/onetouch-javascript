TS600 Series Intelligent PLC

INVT TS600 series intelligent PLC integrates high-performance embedded technology and adopts a high-speed bus system architecture. It combines four types of automation control—sequential, process, information, and motion—within a unified system. Real-time control and complex calculations are achieved through highly reliable hardware and software systems.

The PLC also offers open communication interfaces, IoT networking, and a distributed modular architecture. With fully self-developed programming software and customizable services, programming becomes flexible and user-friendly. When combined with INVT's VFDs, servos, HMIs, and IoT products, the TS600 enables one-stop automation solutions.

Key Features

High Performance

Efficient operation with a 1GHz main frequency, and compiler-based instructions with bit operation speeds up to 0.01μs.

Powerful Motion Control

High-speed motion control for complex processes. Supports 4/8-axis pulse control and EtherCAT motion control.

Easy Connectivity

Multi-protocol support for interconnectivity. Dual Ethernet ports for easy cascading and network isolation.

Easy Programming

Complies with IEC 61131 standards, supporting LD, SFC, IL, and C languages with unified motion control instructions.

Flexible Expansion

CPU equipped with 8/16 digital inputs and 8/16 digital outputs as standard, compatible with Flex series I/O modules.

Cloud Collaboration

Supports VPN remote upload/download, 4G/Wi-Fi/RJ45 transmission methods, and cloud-edge collaboration.

Technical Specifications

Main Frequency 1GHz
Bit Operation Speed Up to 0.01μs
Motion Control 4/8-axis pulse control, EtherCAT motion control
Synchronization 16-axis within 1ms, expandable to 32-axis within 2ms
Digital I/O 8/16 DI, 8/16 DO (standard)
Communication Dual Ethernet ports, Multiple protocols
EtherCAT Stations Up to 72 slave stations
Programming Languages IEC 61131: LD, SFC, IL, C
Connectivity 4G/Wi-Fi/RJ45, VPN, MQTT, API
