TM700 Series High-performance PLC

The TM700 series high-performance programmable controller is mainly designed for scenarios with high motion control requirements and complex control networks. It has significant improvements in control performance, communication capabilities, and programming efficiency, allowing you to build control networks flexibly.

Through OPCUA, the data interaction with the information layer is more convenient, further improving device takt time, shortening development cycles, and bringing an enhanced experience. The TM700 series integrates powerful motion control capabilities with industry-specific process libraries.

Key Features

Powerful Motion Control

1ms@16 axes EtherCAT bus motion control achieving multi-axis collaboration, significantly shortening takt time.

High-Speed I/O

8 high-speed inputs and 8 high-speed outputs integrated, supporting encoder input, hardware latch, and pulse axis functions.

Efficient Network

Rich network interfaces: EtherCAT, ModbusTCP, TCP/IP, UDP, EtherNet/IP, Modbus RTU, and CANopen support.

Easy Programming

Variable usage visibility helps with program optimization and troubleshooting. Axis unit conversions are clear and concise.

Robotic Arms Control

Axis group function with kinematic models integrating multiple standards for various types of robotic arm control.

Industry-Specific Libraries

Integrated with industry-specific process instructions, reducing development difficulty and shortening solution cycles.

Advanced Motion Control

  • BufferMode Function: Smooth motion reducing machinery impact and improving productivity
  • Electronic Cams & Gears: High-order motion control capabilities
  • Multiple Interpolation: Linear, arc, and spiral interpolation functions
  • Precise Position Control: Widely used in labeling and flying trigger scenarios
  • OPC UA Support: Tag communication for efficient information exchange
  • Servo Integration: Host controller reads/writes servo and VFD function codes

Technical Specifications

Motion Control 1ms@16 axes EtherCAT bus control
